# Silverlight.js

CommonJS/AMD wrapper for Microsoft's Silverlight.js

## Install

    bower install tomtwo/silverlight
    npm install tomtwo/silverlight

## Usage

Use like any other CommonJS/AMD package. Otherwise, falls back to window.Silverlight
